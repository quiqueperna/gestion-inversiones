# Estilo

Toda la información debe ser en español.

* **Estilo:** Dashboard con estética Dark Mode Moderna (Glassmorphism sutil)
* **Tipografía:** CSS/Tailwind
* Títulos Secciones Sans Serif 0.75rem (12px) 700 (Bold) 
* Valores Destacados Sans Serif 1.25rem (20px) 600 (Semibold)
* Cuerpo de Tabla Sans Serif 0.875 rem (14px) 400 (Regular)
* Nombres de Ítems Sans Serif0.875 rem (14px) 800 (Extra Bold)
* El espaciado entre letras (letter-spacing) en los títulos en mayúsculas es tracking-wider en Tailwind.

* **Layout:** Interfaz de alta densidad con filas compactas (altura <40px) para maximizar datos visibles.
* Tipografía (Sans-Serif): * Títulos/Headers: 10-11px, Bold, Uppercase, color gris tenue.
* Datos/Cuerpo: 13-14px, Medium/Bold para ítems clave (ej. Tickers o Montos).
* Componentes:
  * Filtros: Botones tipo "Pill" compactos; estado activo en azul vibrante.
  * Tablas: Celdas con padding vertical mínimo (py-1 o py-2).
  * Semántica: Verde esmeralda para rendimientos positivos y naranja/rojo para alertas de stock o críticas.
