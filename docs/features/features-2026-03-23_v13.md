## 1. CashFlowForm — Cuentas dinámicas

**Pantalla:** Nuevo Depósito / Retiro
**Problema resuelto:** El combo "Cuenta" mostraba una lista hardcodeada (`["USA", "Argentina", "CRYPTO"]`). Las cuentas creadas desde el CRUD de Cuentas no aparecían.

**Implementación:**
- `CashFlowFormProps` ahora acepta prop opcional `cuentas?: string[]`
- Si se pasa la prop y tiene elementos, se usa esa lista; si no, fallback a `DEFAULT_CUENTAS`
- `page.tsx` pasa `cuentas={cuentas.map(c => c.nombre)}` al `<CashFlowForm>`

**Archivos modificados:**
- `src/components/cashflow/CashFlowForm.tsx`
- `src/app/page.tsx`
