# Features — Sesión v21 (2026-03-23)

## Importación CSV con generación de trades y todas las estrategias

### Descripción
Motor de simulación de trades para importación masiva de ejecuciones. Los trades se generan automáticamente respetando la estrategia configurada en cada cuenta, con soporte completo de FIFO, LIFO, MAX_PROFIT, MIN_PROFIT y MANUAL.

### Motor de simulación (`trade-simulator.ts`)
- Input: filas a importar + executions abiertas en DB + configuración de cuentas + decisiones manuales
- Procesa en orden cronológico
- BUY → entra al inventario → genera ProjectedTrade OPEN
- SELL → busca contraparte según estrategia de la cuenta → genera ProjectedTrade CLOSED
- Cross-import: un BUY y SELL del mismo lote se pueden matchear entre sí
- Cierre parcial: si BUY tiene más qty que SELL, el resto queda OPEN

### Estrategias
| Estrategia | Lógica |
|---|---|
| FIFO | Cierra el BUY más antiguo primero |
| LIFO | Cierra el BUY más reciente primero |
| MAX_PROFIT | Cierra el BUY con mayor PnL estimado |
| MIN_PROFIT | Cierra el BUY con menor PnL estimado |
| MANUAL | Usuario elige el BUY en un flow interactivo |

### Flujo UI (5 pasos)
1. **upload**: carga archivo, preview de filas válidas/inválidas, botón "Calcular trades"
2. **preview**: secciones colapsables — ejecuciones + trades proyectados. Si hay MANUAL → botón para resolver
3. **manual**: una a una, para cada SELL con estrategia MANUAL: muestra candidatos BUY con PnL estimado, usuario elige
4. **final-review**: preview con todas las decisiones incluidas, botón "Confirmar"
5. **done**: banner de éxito con totales

### Server actions
- `previewBulkImport(rows, manualDecisions?)` → SimulationResult (sin escribir DB)
- `confirmBulkImportWithTrades(rows, manualDecisions)` → crea Executions + TradeUnits en DB
