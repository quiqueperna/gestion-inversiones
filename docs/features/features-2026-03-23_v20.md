# Features — Sesión v20 (2026-03-23)

## Importación CSV — Validación de broker/cuenta + dos tablas separadas

### Descripción
Extensión de la importación CSV (v19) con validación de existencia de broker y cuenta en BD, y UI con dos listados diferenciados.

### Archivos modificados
| Archivo | Cambio |
|---|---|
| `src/lib/csv-import.ts` | Parámetros `validBrokers?` y `validAccounts?`; errores específicos por campo |
| `src/components/trades/ImportCSVView.tsx` | Dos tablas: válidas + inválidas con columna Razón; props brokers/accounts |
| `src/app/page.tsx` | Pasa `brokers` y `accounts` a `ImportCSVView` |

### Lógica de validación
Una fila es inválida si:
- Fecha no tiene formato `mm/dd/yy hh:mm:ss`
- Side no es BUY o SELL
- Qty no es número válido
- Precio no es número positivo
- Symbol está vacío
- **Broker no existe en la BD** → `"Broker 'X' no existe"`
- **Cuenta no existe en la BD** → `"Cuenta 'X' no existe"`

Una fila puede tener múltiples razones (separadas por ` · `).

### UX
- Tabla verde: "N ejecuciones listas para importar"
- Tabla roja: "N filas que NO se importarán" con columna Razón al final
- Botón confirmar deshabilitado si 0 filas válidas
