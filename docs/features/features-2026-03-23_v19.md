# Features — Sesión v19 (2026-03-23)

## Importación masiva de ejecuciones desde CSV

### Descripción
Nueva funcionalidad que permite cargar un archivo CSV/TSV con ejecuciones de operaciones y previsualizarlas antes de confirmar la importación a la base de datos.

### Archivos creados/modificados
| Archivo | Acción |
|---|---|
| `src/lib/csv-import.ts` | NUEVO — parser CSV/TSV cliente |
| `src/components/trades/ImportCSVView.tsx` | NUEVO — UI de importación con preview |
| `src/server/actions/trades.ts` | MODIFICADO — `bulkImportExecutions()` |
| `src/app/page.tsx` | MODIFICADO — view "importar-csv" + botón toolbar |

### Formato CSV soportado
```
Exec Time       Side  Qty  Symbol  Net Price  Broker  Account
12/27/22 13:15:15  BUY   +30  NIO     9.9999     Schwab  USA
```
- Separador: tab (TSV) o coma (CSV) — autodetectado
- Fecha: `mm/dd/yy hh:mm:ss`
- Qty: acepta prefijo `+`

### Flujo de usuario
1. Click "Importar CSV" en toolbar
2. Drag & drop o selección de archivo
3. Preview table con todas las filas (errores marcados en rojo)
4. "Confirmar N ejecuciones" → importación a BD
5. Banner de éxito → Cerrar → recarga de datos
