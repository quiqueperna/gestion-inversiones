# Bitácora de Prompt — 2026-03-23 v13

## Prompt recibido

```
Pantalla Nuevo deposito
- Agregué una nueva cuenta y cuando abro esta pantalla no me aparece en el combo "cuentas" la cuenta recién agregada

Pantalla Dashboard
- En la tarjeta Balance Total cambiar nombre de "Trade Units abiertas" por "Trades Abiertos"
```

## Análisis

**Problema 1 (CashFlowForm):**
Root cause: `CashFlowForm.tsx` tenía `const CUENTAS = ["USA", "Argentina", "CRYPTO"]` hardcodeado. No recibía las cuentas del estado global de `page.tsx`.

**Problema 2 (Dashboard):**
Root cause: texto literal incorrecto en `page.tsx` línea 860.

## Solución implementada

1. Agregué prop `cuentas?: string[]` a `CashFlowFormProps` con fallback a `DEFAULT_CUENTAS`
2. Pasé `cuentas={cuentas.map(c => c.nombre)}` desde `page.tsx`
3. Cambié label en dashboard
