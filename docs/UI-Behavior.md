## Listado de Operaciones

**El campo de precio de salida** es el valor que toma el precio cuando cerramos la operacion. Si la operacion
no está cerrada debe tomar el precio actual con el simbolo de la operacion con googleFinance (preferentemente) o con Yahoo finance.
**Fecha de Salida**  es la fecha que cerramos la operacion, sino esta cerrada es la fecha de hoy (dd/mm/aaaa)

## Funcionalidad
El listado debe mostrar todas los campos de las operaciones.
