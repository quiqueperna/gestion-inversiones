# Misión: Agente UI — v19 (2026-03-23)

## Objetivo
Pantalla de importación masiva de ejecuciones CSV con preview.

## Componente: `src/components/trades/ImportCSVView.tsx`
- Prop `onImport(rows): Promise<void>` — callback al confirmar
- Prop `onClose(): void`
- Prop `inline?: boolean`

### UI:
1. **Zona de carga**: drag-and-drop + click para seleccionar archivo .csv/.txt/.tsv
2. **Preview table**: muestra todas las filas parseadas con columnas Date, Side, Qty, Symbol, Price, Broker, Account
   - Filas con `error` se marcan en rojo
   - Contador: "N ejecuciones listas para importar"
3. **Botones**: Cancelar | Confirmar importación (disabled si 0 filas válidas)
4. **Estado**: loading spinner durante importación

### Estilo: dark glassmorphism, mismas clases que el resto de la app

## Integración en `src/app/page.tsx`
