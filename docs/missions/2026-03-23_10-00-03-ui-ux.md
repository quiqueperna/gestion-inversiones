## Tarea 1: CashFlowForm — cuentas dinámicas

**Problema:** `CashFlowForm.tsx` usa `const CUENTAS = ["USA", "Argentina", "CRYPTO"]` hardcodeado.
Las cuentas creadas vía CRUD no se reflejan en el combo.

**Solución:**
- Agregar prop `cuentas?: string[]` a `CashFlowFormProps`
- Reemplazar uso de `CUENTAS` hardcodeado por `cuentas ?? CUENTAS` (fallback)
- En `src/app/page.tsx`, pasar `cuentas={cuentas.map(c => c.nombre)}` al `<CashFlowForm>`

**Archivos a modificar:**
- `src/components/cashflow/CashFlowForm.tsx`
- `src/app/page.tsx`
